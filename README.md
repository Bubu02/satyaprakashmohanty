# Portfolio Website

This is a personal portfolio website recreated from Bhuvaneshwari Basquarane's portfolio.

## Deployment to GitHub Pages

1.  Push this entire directory to a GitHub repository (e.g., named `yourusername.github.io`).
2.  Go to **Settings** > **Pages**.
3.  Select the **Source** as `Deploy from a branch`.
4.  Select the branch `main` and folder `/ (root)`.
5.  Click **Save**.

Your site will be live in a few minutes!
